<h2>
            &nbsp;&nbsp;&nbsp;&nbsp;O QUE É?
</h2>

<br>

<p>
            &nbsp;&nbsp;&nbsp;&nbsp;O <strong>Gunter</strong> é um bot para o Discord desenvolvido puramente com fins educacionais. O objetivo deste projeto é colocar a teoria em prática, pois acredito que a melhor forma de aprender é colocando a mão na massa: desenvolver uma aplicação real, funcional e interativa acelera o aprendizado e traz desafios extremamente valiosos para a bagagem de quem está aprendendo a programar.
</p>

<br>

<br>

> [!Note]
>      "**Gunter**" é o nome do icônico pinguim do desenho animado "**Hora de Aventura**"
